# Dynamic Pricing with Random Forest

Bu proje, perakende satis verileri uzerinde Random Forest tabanli bir dynamic pricing prototipi gelistirir. Repo icinde veri hazirlama, feature engineering, model egitimi ve profesyonel bir Streamlit dashboard bulunur.

## Dashboard

Dashboard su ekranlari sunar:

- yonetici seviyesi KPI kartlari
- kategori ve musteri segment analizi
- model performans ozeti
- canli tahmin ve simulasyon paneli

## Calistirma

```bash
pip install -r requirements.txt
python -m streamlit run app.py
```

## Teknolojiler

- pandas
- scikit-learn
- Plotly
- Streamlit
